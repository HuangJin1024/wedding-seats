# 婚礼宴席排座系统 - 规格说明书

## 1. 功能需求

### 1.1 宾客管理
- 添加宾客（姓名、手机号、备注）
- 编辑宾客信息
- 删除宾客
- 搜索宾客
- 批量导入（可选）

### 1.2 桌位管理
- 创建桌号（1, 2, 3... 或自定义如"A桌"）
- 设置每桌容纳人数
- 拖拽宾客到桌位
- 查看每桌已坐/剩余座位

### 1.3 座位安排
- 将宾客拖放到对应桌位
- 同桌宾客可显示关联关系
- 未安排座位的宾客显示在"待安排"列表

### 1.4 数据持久化
- 自动保存到 localStorage
- 支持导出/导入 JSON

## 2. 界面设计

### 2.1 顶部
- 标题：婚礼宴席排座
- 总宾客数 / 已安排数显示

### 2.2 左侧 - 宾客列表
- 待安排宾客（可拖拽）
- 搜索框

### 2.3 右侧 - 桌位区域
- 桌位卡片网格显示
- 每张桌显示：桌号、宾客名字列表、剩余座位
- 可拖拽宾客到桌位

### 2.4 底部/弹窗
- 添加宾客按钮
- 添加桌位按钮
- 导入/导出按钮

## 3. 技术实现

- 纯 HTML + CSS + JavaScript
- 拖拽 API 实现拖放
- localStorage 本地存储
- 单文件，方便复制分享
